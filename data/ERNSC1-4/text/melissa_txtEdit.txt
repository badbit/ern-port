CERT Advisory CA-99-04-Melissa-Macro-Virus

Sistemas Afectados

     * Máquinas con Microsoft Word 97 o Word 2000
     * Cualquier sistema de manejo de correo puede experimentar problemas
       en el desempeño como resultado de la propagación de este macro virus.

Revisión

   A las 2:00 PM GMT-5 aprox. del Viernes 26 de Marzo de 1999 empezamos
   a recibir reportes de un Macro Virus de Microsoft Word 97 y Word 2000
   que se estaba propagando via attachments. El número y variedad de los
   reportes que recivimos indican que es un ataque diseminado afectando
   a una gran variedad de sitios.

   Nuestros análisis de este macro virus indican que acción humana (en la
   forma de un usuario abriendo un documento de Word infectado) es requrido
   por este virus para propagarse. Es posible que bajo algunas
   configuraciones, algún usuario automágicamente abra un documento infectado
   recibido en la forma de un attachment de email. No se sabe si este virus
   explote nuevas vulnerabilidades. Aunque el mecanismo de transporte
   primario de este virus es vía email, cualquier forma de tranmisión de
   archivos también puede propagar el virus.

   Los vendedores de Anti-virus han llamado a este macro virus el macro
   Melissa o el virus W97M_Melissa.

I. Descripción

   El virus Melissa se propaga en la forma de un mensaje de email
   conteniendo en documento de Word infectado como attachment. El mensaje
   portador has sido frecuentemente reportado por tener el siguiente
   Subject

      Subject: Important Message From <nombre>

   Donde <nombre> es el nombre completo de la persona que envió el mensaje.

   El cuerpo del mensaje es un mensaje MIME multipartido conteniendo dos
   seciones. La primera sección del mensaje (Content-Type: text/plain)
   contiene el siguiente texto.

      Here is that document you asked for ... don't show anyone else ;-)

   La siguiente sección (Content-Type: application/msword) fué inicialmente
   reportada de ser un documento llamado "list.doc". Este documento contiene
   referencias a sitios web pornográficos. Mientras este virus se propaga
   esperamos ver documentos con otros nombres. De hecho, bajo ciertas
   condiciones el virus puede generar attachments con documentos creados
   por la víctima.

   Cuando el usuario abre un archivo .doc infectado con Microsoft Word97 o
   Word2000, el macro virus es inmediatamente ejecutado si macros están
   habilitados.

   Después de la ejecución, el virus primero baja las cofiguraciones de
   seguridad de macros security para permitir que todos los macros
   corran cuendo documentos sean abiertos en el futuro. Por lo tanto, el
   usuario no será notificado de cuando el virus sea ejecutado en el
   futuro.

   El macro checa el registro para ver si la clave

   "HKEY_Current_User\Software\Microsoft\Office\Melissa?"

   tiene el valor de "... by Kwyjibo". Si la clave del registro no existe
   o no tiene el valor de "... by Kwyjibo", el virus procede a
   propagarse mandando un email con el fomato descrito arriba a las primeras
   50 entradas en cualquier libreta de direcciones MAPI legibles por 
   el usuario ejecutando el macro. Mantén en mente que si cualquiera de esas
   direcciones son listas de correo, el mensaje será entregado a todos
   en la lista. En orden para propagarse con éxito, la máquina afectada
   debe tener Microsoft Outlook instalado; de todos modos, Outlook no
   necesariamente tiene que ser el mailer usado para leer el mensaje.
